Perfil Profesional

Desarrollador de software con experiencia en la creación de aplicaciones web modernas, diseño de APIs y desarrollo backend. Actualmente trabajo con Laravel y React, construyendo soluciones funcionales enfocadas en optimizar procesos como inventarios, ventas, control administrativo y automatización de tareas repetitivas. Tengo experiencia diseñando arquitecturas limpias, estructurando bases de datos relacionales y creando sistemas a medida para pequeñas y medianas empresas. Me encuentro fortaleciendo mi perfil backend al profundizar en Java con Spring Boot, y ampliando mi stack hacia Angular, con el objetivo de desarrollar aplicaciones más escalables, modulares y robustas. Mi enfoque es práctico: aprendo construyendo, mejoro iterando y priorizo soluciones que resuelvan problemas reales de negocio. Soy autodidacta, adaptable y persistente. Disfruto entender cómo funciona cada pieza del sistema, desde el análisis inicial hasta la API que consume el frontend. Mi meta es consolidarme como un desarrollador backend fuerte, capaz de crear software de calidad con buenas prácticas, arquitectura sólida y visión a largo plazo.
